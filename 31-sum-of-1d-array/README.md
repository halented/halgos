# Running Sum of 1D array

Given an array nums. We define a running sum of an array as runningSum[i] = sum(nums[0]â€¦nums[i]).

Return the running sum of nums.

## Examples

```javascript
    runningSum([1,2,3,4])
        => [1,3,6,10]
    runningSum([1,1,1,1,1])
        => [1,2,3,4,5]
```

## Constraints
- `1 <= nums.length <= 1000`
- `-10^6 <= nums[i] <= 10^6`
