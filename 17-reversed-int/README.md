# Reverse Integer

Given a 32-bit signed integer, reverse and return its digits.

## Examples

```javascript
    reverse(123)
        => 321
    reverse(0)
        => 0
    reverse(-123)
        => -321
```

## Constraints
- can only output integers within the 32-bit signed integer range: [−2^31,  2^31 − 1].
