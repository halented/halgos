# Find the Nth Fibonacci Number

Given a number *n*, return the related Fibonacci number.

Examples:
```javascript
fib(0)  # => 0
fib(1)  # => 1
fib(2)  # => 1
fib(3)  # => 2
fib(4)  # => 3
```
